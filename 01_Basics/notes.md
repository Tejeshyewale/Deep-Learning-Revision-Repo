# Basics of Deep Learning

## 🔹 Perceptron
- Single neuron model
- Output = activation(w.x + b)

## 🔹 Activation Functions
- ReLU → max(0, x)
- Sigmoid → 1 / (1 + e^-x)
- Tanh → range (-1,1)

## 🔹 Gradient Descent
- Used to minimize loss
- Update rule:
    w = w - lr * gradient

## 🔹 Loss Functions
- MSE → Regression
- Cross Entropy → Classification
