# Neural Networks

## 🔹 What is a Neural Network?

A Neural Network is a collection of layers of neurons where each neuron performs:
output = activation(weight * input + bias)

---

## 🔹 Types of Layers

* Input Layer → Takes input features
* Hidden Layer → Performs computations
* Output Layer → Produces final result

---

## 🔹 Forward Propagation

* Data flows from input → hidden → output
* Each layer applies weights, bias, and activation

---

## 🔹 Backpropagation

* Used to update weights
* Works using chain rule
* Minimizes loss function

---

## 🔹 Loss Functions

* MSE → Regression
* Cross Entropy → Classification

---

## 🔹 Key Concepts

* Weights & Bias
* Activation Functions
* Learning Rate
* Epochs & Batch Size

---

## 🔹 Overfitting & Underfitting

* Overfitting → Model memorizes data
* Underfitting → Model fails to learn

👉 Solution:

* Regularization
* Dropout
* More data
